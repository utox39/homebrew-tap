# Utox39 Tap

## How do I install these formulae?

`brew install utox39/tap/<formula>`

Or `brew tap utox39/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "utox39/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
